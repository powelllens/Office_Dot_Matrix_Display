# Office_Dot_Matrix_Display

All software, description and required steps that are created during the development of the Office Dot Matrix Display

The Display is buildup with six 32x32 RGB LED Dot Matrix Display's in series, controlled by Raspberry Pi 2. The Library that ist used to control and handle the Display panels is from hzeller/rpi-rgb-led-matrix and can be found under:

<https://github.com/hzeller/rpi-rgb-led-matrix>
